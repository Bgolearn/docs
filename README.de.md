# BGOlearn-Algorithmushandbuch

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Deutsch](README.de.md)

Dieses Repository hostet die statische Dokumentationswebsite von [BGOlearn](https://github.com/BGOlearn). Sie enthält Anleitungen zur bayesschen globalen Optimierung, zu Optimierungsstrategien, Surrogatmodellen, Akquisitionsfunktionen und praktischen Beispielen.

## Online-Dokumentation

Die Dokumentation ist über die folgenden fortlaufend gepflegten Endpunkte verfügbar:

- **GitHub Pages (empfohlen):** <https://bgolearn.github.io/docs/>
- **Netlify:** <https://bgolearn.netlify.app/>

Beide Endpunkte stellen dieselbe Dokumentation bereit. GitHub Pages wird als alternativer Zugangsweg gepflegt, damit Nutzer in Ländern oder Netzwerken, in denen `netlify.app` nicht erreichbar ist, weiterhin auf das BGOlearn-Algorithmushandbuch zugreifen können.
