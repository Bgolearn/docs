# BGOlearn 알고리즘 매뉴얼

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Deutsch](README.de.md)

이 저장소는 [BGOlearn](https://github.com/BGOlearn)의 정적 문서 웹사이트를 관리합니다. 베이지안 전역 최적화, 최적화 전략, 대리 모델, 획득 함수 및 실습 예제를 제공합니다.

## 온라인 문서

문서는 지속적으로 관리되는 다음 주소에서 이용할 수 있습니다.

- **GitHub Pages(권장):** <https://bgolearn.github.io/docs/>
- **Netlify:** <https://bgolearn.netlify.app/>

두 주소는 동일한 문서를 제공합니다. 일부 국가 또는 네트워크 환경에서 `netlify.app`에 접근할 수 없는 사용자도 BGOlearn 알고리즘 매뉴얼을 볼 수 있도록 GitHub Pages를 대체 접속 경로로 유지합니다.
