本仓库托管 [BGOlearn](https://github.com/BGOlearn) 的静态文档网站，内容涵盖贝叶斯全局优化、优化策略、代理模型、采集函数及实践示例。

## 在线文档

文档可通过以下持续维护的地址访问：

- **GitHub Pages（推荐）：** <https://bgolearn.github.io/docs/>
- **Netlify：** <https://bgolearn.netlify.app/>

两个地址提供相同的文档内容。GitHub Pages 作为备用访问入口持续维护，方便部分国家或网络环境中无法访问 `netlify.app` 的用户阅读 BGOlearn 算法手册。
